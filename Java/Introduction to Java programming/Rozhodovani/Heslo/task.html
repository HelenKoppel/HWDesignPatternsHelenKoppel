Heslo

Uloz si do promenne typu String libovolne heslo.
V tomto ukolu se budeme rozhodovat jesli uživatel zadal spravne heslo.
Vyžádej si od uživatele heslo na základě toho jestli je rovno s tvou promennou vypiš výsledek
"Pristup je povolen"
Jestli ne:
"Pristup zamezen!"
