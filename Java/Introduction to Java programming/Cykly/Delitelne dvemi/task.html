Delitelne Dvouma

Tvojim ukolem bude vytvorit cyklus for, ktery bude pripocitavat cislo od 0 nahoru.
Cyklus se musi zastavit kdyz cislo bude mit hodnotu 50.
Vypis jenom parne cisla.

Jak zjistit ze je cislo parne?
Kazde parne cislo je beze zbytku delitelne 2.
Podobne jako mame operaci deleni za pomoci lomitka /
mame i operaci ktera zjisti zustatek po deleni a to operace modulo
ktera se zapisuje za pomoci symbolu %
Napriklad:
3 % 2 = 1;
4 % 2 = 0;
5 % 3 = 2;

Takze jestli je cislo parne zjistime nasledovne:
int cislo = 5;
int zbytekPoDeleni = cislo % 2;
boolean jeParni = zbytekPoDeleni == 0;

Vystup naseho programu by mel vypadat nasledovne:
0
2
4
...
50

for(int i = 0; i < ...; i = i + 1) {
  if (jeParni) {
    System.out.println(cislo);
  }
}
