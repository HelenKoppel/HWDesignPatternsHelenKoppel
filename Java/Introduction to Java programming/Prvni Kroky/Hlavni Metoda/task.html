Hlavní Metoda!

V této lekci je tvým úkolem upravit program tak, obsahoval metodu : main

K řešení si budeš muset vzpomenout, jak přesně vypadá hlavní metoda v Javě. Jestli si nevzpomeneš, tak je pro tebe níže připravena nápověda ;).

Za úkolem nehledej žádnou vědu, opravdu stačí jen vytvořit metodu a kontrola projde!
