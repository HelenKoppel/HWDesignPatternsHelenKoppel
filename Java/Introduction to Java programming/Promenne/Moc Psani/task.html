Moc Psaní

V minulé lekci jsme si dali moc práce s přepisem jedné hodnoty. Teď můžeme využít proměnné.
Vytvoř novou proměnnou typu String a ulož do ní značku výrobce automobilů "Bentley".
Následně využij při vypisování daného textu proměnnou.

K spojení dvou textů se používá symbol +
Tedy:
"ahoj, ja jsem " + "Peter"
se bude rovnat:
"ahoj, ja jsem Peter"

Vypiš věty uvedené níže za pomoci proměnné:

  • Bentley je výrobcem automobilů
  • Bentley je symbolem luxusu
  • Bentley je britská značka
String mojePromenna = "Olomouoc";
System.out.println(mojePromenna + " je krasne mesto");

Jakmile bude tvůj úkol hotov, zkus experimentovat s různými hodnotami proměnné a jak jednoduché je teď měnit velký kus textu.
